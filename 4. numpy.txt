import numpy as np

# Create a 5-dimensional array
arr = np.array([1, 2, 3, 4], ndmin=5)

# Display the array and verify its dimensions
print("Array:")
print(arr)
print("Number of dimensions:", arr.ndim)
Algorithm:

Import the NumPy module.

Create an array using np.array() with ndmin=5 to make it 5D.

Print the array.

Use .ndim to check and print the number of dimensions.

import numpy as np

# Display the result
print("Original Array:", arr)
print("Sorted Array:", sorted_arr)
