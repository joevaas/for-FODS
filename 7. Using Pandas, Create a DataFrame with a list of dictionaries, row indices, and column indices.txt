Q7. (i) Using Pandas, Create a DataFrame with a list of dictionaries, row indices, and column indices
Aim:

To create a DataFrame from a list of dictionaries with row and column labels.

Algorithm:

Import Pandas.

Create a list of dictionaries.

Specify row and column indices.

Create a DataFrame using pd.DataFrame().

Program:
import pandas as pd

data = [{'a': 10, 'b': 20}, {'a': 30, 'b': 40, 'c': 50}]
df = pd.DataFrame(data, index=['row1', 'row2'], columns=['a', 'b', 'c'])
print(df)

Output:
        a     b     c
row1  10.0  20.0   NaN
row2  30.0  40.0  50.0

Result:

A DataFrame was created from list of dictionaries with specified indices.

Q7. (ii) Use index label to delete or drop rows from a Pandas DataFrame
Aim:

To delete rows using index labels.

Algorithm:

Import Pandas.

Create a DataFrame.

Use drop() with index label to remove rows.

Program:
import pandas as pd

data = {'Name': ['Ravi', 'Kumar', 'Sita'], 'Age': [22, 23, 21]}
df = pd.DataFrame(data, index=['r1', 'r2', 'r3'])

print("Before Deletion:\n", df)
df = df.drop('r2')
print("\nAfter Deletion:\n", df)

Output:
Before Deletion:
      Name  Age
r1   Ravi   22
r2  Kumar   23
r3   Sita   21

After Deletion:
     Name  Age
r1  Ravi   22
r3  Sita   21

Result:

Row deleted successfully using index label.
