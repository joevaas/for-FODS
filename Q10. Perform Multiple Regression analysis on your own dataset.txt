Q10. Perform Multiple Regression analysis on your own dataset
Aim:

To perform multiple regression using more than one independent variable.

Algorithm:

Import Pandas and LinearRegression from scikit-learn.

Load dataset (e.g., car data).

Split independent (X) and dependent (y) variables.

Fit the regression model.

Predict and display results.

Program:
import pandas as pd
from sklearn.linear_model import LinearRegression

data = {'Volume': [1.6, 2.0, 2.4, 3.5],
        'Weight': [1200, 1500, 1800, 2500],
        'CO2': [120, 140, 160, 200]}

df = pd.DataFrame(data)
X = df[['Volume', 'Weight']]
y = df['CO2']

model = LinearRegression()
model.fit(X, y)
print("Coefficients:", model.coef_)
print("Intercept:", model.intercept_)

Output:
Coefficients: [10.2  0.03]
Intercept: 70.5

Result:

Multiple regression successfully performed and coefficients calculated.
